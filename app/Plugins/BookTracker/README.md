## Konfigurasi

- **Target Buku/Tahun**: Berapa buku ingin Anda baca tahun ini (default: 12)
- **Pengingat Membaca Harian**: Aktifkan pengingat untuk membaca
- **Waktu Pengingat**: Kapan Anda ingin diingatkan (default: 20:00)
- **Hari Pengingat**: Pilih hari mana saja (default: setiap hari)
- **Ringkasan Bulanan**: Terima summary progress bulanan
- **Lacak Genre**: Track genre buku yang dibaca
- **Simpan Kutipan Favorit**: Fitur menyimpan quotes dari buku
- **Bagikan Progress**: Share achievement membaca Anda

## Cara Menggunakan

1. Aktifkan plugin dan set target membaca tahunan
2. Catat buku yang Anda baca via chat dengan ASPRI
3. Update progress membaca (halaman/chapter yang sudah dibaca)
4. Simpan kutipan favorit dari buku
5. Review dan rate buku setelah selesai
6. Lihat statistik dan progress menuju target

## Target Realistis

- **12 buku/tahun** = 1 buku/bulan (pemula)
- **24 buku/tahun** = 2 buku/bulan (moderate reader)
