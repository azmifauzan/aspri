## Konfigurasi

- **Lacak Berat Badan**: Aktifkan tracking berat badan
- **Lacak Langkah Kaki**: Aktifkan tracking langkah kaki
- **Target Langkah/Hari**: Berapa langkah target Anda per hari (default: 10.000)
- **Lacak Air Minum**: Aktifkan tracking konsumsi air
- **Target Air/Hari**: Berapa gelas air target per hari (default: 8)
- **Lacak Jam Tidur**: Aktifkan tracking jam tidur
- **Target Tidur/Hari**: Berapa jam tidur ideal (default: 8)
- **Waktu Pengingat Harian**: Kapan Anda ingin diingatkan untuk mencatat
- **Laporan Mingguan**: Terima laporan progress mingguan
- **Hari Laporan Mingguan**: Hari apa laporan dikirim

## Cara Menggunakan

1. Aktifkan plugin dan konfigurasikan metrik yang ingin dilacak
2. Setiap hari Anda akan menerima pengingat untuk mencatat data kesehatan
3. Catat data Anda dengan chat ke ASPRI (contoh: "Hari ini jalan 12.000 langkah")
4. Terima laporan mingguan untuk melihat progress Anda

## Tips

- Konsisten mencatat setiap hari untuk hasil terbaik
- Set target yang realistis dan tingkatkan secara bertahap
- Gunakan laporan mingguan untuk evaluasi progress
